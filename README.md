# Uni-Ui

UI components for react.js websites and apps

## Install

You can install Uni-UI using either of the methods given below.

For npm users:

```shell
  npm i @web-uni/uni-ui
```

For Yarn users:

```shell
  yarn add @web-uni/uni-ui
```
