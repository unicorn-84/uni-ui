# Text

Arbitrary text.

## Usage

```javascript
import { Text } from '@web-uni/uni-ui';
<Text>text</Text>
```

## Properties

### children

Anything that can be rendered.

- *_Required_*

```text
node
```

### className

Custom classes.

```text
string
```

### style

Custom style object.

```text
object
```

### a11yLabel

Custom label to be used by screen readers. When provided, an aria-label will be added to the element.

```text
string
```

## Intrinsic element

span
