# Text

Arbitrary text.

## Usage

```javascript
import { Text } from '@web-uni/uni-ui';
<Text>Arbitrary text</Text>
```

## Properties

### style

Custom JavaScript object for inline CSS.

```text
object
```

### a11yLabel

Custom label to be used by screen readers. When provided, an aria-label will be added to the element.

```text
string
```
